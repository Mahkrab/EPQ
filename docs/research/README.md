# Maelstrom EPQ Research Sources

Research record.

- `source-register.md` is the readable overview.
- `references.bib` is the citation database for the report.
- `sources/` contains one focused evaluation record per source.
- `sources/TEMPLATE.md` is the canonical format for new source records.

## Adding a source

1. Copy `sources/TEMPLATE.md` into a new file named with the next source number, for example `014-short-source-title.md`.
2. Replace `S###` with the matching source ID.
3. Add a row for the source in `source-register.md`.
4. Add a BibTeX entry to `references.bib` if the source will be cited in the report.

Keep entries concise. The source record should explain relevance and reliability, not become a general reading diary.
