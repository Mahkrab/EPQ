# S018 — OpenBSD Project (n.d.)

## Full reference:

OpenBSD Project (n.d.) ‘OpenSSH’, *OpenSSH*.

#### https://www.openssh.org/

## Relevance:

Provides authoritative technical background for the OpenSSH server used to expose the Ubuntu WSL 2 development environment securely. It identifies the client and server tools, encryption, authentication methods, tunnelling and configuration capabilities relevant to the remote-development setup.

## Reliability:

This is the official website of OpenSSH, maintained by its developers within the OpenBSD Project. It is a primary source for the software’s purpose and components, although detailed configuration decisions should also be checked against the manual page for the installed OpenSSH version.

## Tags

`networking`, `openssh`, `ssh-server`, `remote-access`

## Reading notes

- OpenSSH provides the `ssh` client and the server-side `sshd` daemon.
- It encrypts remote-login traffic and supports several authentication methods.
- The suite also includes secure file-transfer, key-management and tunnelling tools.
