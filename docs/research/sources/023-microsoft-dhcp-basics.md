# S023 — Microsoft (2022)

## Full reference:

Microsoft (2022) ‘DHCP (Dynamic Host Configuration Protocol) Basics’, *Microsoft Learn*.

#### https://learn.microsoft.com/en-us/windows-server/troubleshoot/dynamic-host-configuration-protocol-basics

## Relevance:

Explains why the Raspberry Pi’s LAN address could change after network maintenance. It supports the diagnosis that DHCP dynamically reassigned IP configuration.

## Reliability:

Microsoft Learn is official technical documentation. The page describes DHCP as a standard protocol and points to its defining RFC, making it suitable for this basic networking claim.

## Tags

`networking`, `dhcp`, `dynamic-address`, `ip-address`

## Reading notes

- DHCP distributes IP addresses and other network configuration to clients dynamically.
- A changed DHCP assignment can make a device unavailable at its previous address.
