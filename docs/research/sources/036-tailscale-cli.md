## Relevance:

Documents the `tailscale status` and `tailscale ping` commands used in the layered diagnostic method to inspect peers and test Tailscale connectivity before testing SSH.

## Reliability:

This is official Tailscale command-line documentation and is authoritative for the current client commands. Command behaviour may vary with installed client versions.

## Tags

`tailscale`, `cli`, `status`, `ping`

## Reading notes

- `tailscale status` reports connections to other devices in the tailnet.
- `tailscale ping` tests connectivity to another Tailscale device.
