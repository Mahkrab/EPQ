# S029 — Cloudflare Inc. (n.d.)

## Full reference:

Cloudflare Inc. (n.d.) ‘What is my IP address?’, *Cloudflare Learning Center*.

#### https://www.cloudflare.com/learning/dns/glossary/what-is-my-ip-address/

## Relevance:

Defines IP addresses and distinguishes IPv4, IPv6, static and dynamic addressing. It supports the README’s explanation of address changes and route selection.

## Reliability:

Cloudflare is an established Internet infrastructure provider. Its Learning Center is an appropriate secondary source for introductory IP terminology, although it is not the protocol standards body.

## Tags

`networking`, `ip-address`, `ipv4`, `ipv6`

## Reading notes

- IP addresses identify networked devices.
- IPv4 and IPv6 use different address formats.
- IP addresses may be assigned statically or dynamically.
