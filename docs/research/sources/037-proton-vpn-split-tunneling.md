# S037 — Proton AG (n.d.)

## Full reference:

Proton AG (n.d.) ‘How to use split tunneling’, *Proton VPN Support*.

#### https://protonvpn.com/support/protonvpn-split-tunneling

## Relevance:

Explains the Proton VPN split-tunnelling controls used during troubleshooting, including application and IP-address include or exclude rules.

## Reliability:

This is official support documentation from Proton, so it is authoritative for the current Proton VPN application. It is vendor-authored and may change as the application is updated.

## Tags

`proton-vpn`, `vpn`, `split-tunneling`, `routing`

## Reading notes

- Split tunnelling can route selected applications or IP addresses inside or outside the VPN tunnel.
- Proton VPN supports include and exclude modes on Windows.
