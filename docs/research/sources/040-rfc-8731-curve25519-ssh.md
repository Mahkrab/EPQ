# S040 — Adamantiadis, Josefsson and Baushke (2020)

## Full reference:

Adamantiadis, A., Josefsson, S. and Baushke, M. (2020) *Secure Shell (SSH) Key Exchange Method Using Curve25519 and Curve448*. RFC 8731. RFC Editor.

#### https://www.rfc-editor.org/info/rfc8731/

## Relevance:

Defines the Curve25519 key-exchange method used in the final SSH client configuration and supports the distinction between this method and the larger hybrid post-quantum exchange.

## Reliability:

This is an Internet Standards Track RFC produced through IETF review and published by the RFC Editor. It is authoritative for using Curve25519 key exchange in SSH.

## Tags

`ssh`, `curve25519`, `key-exchange`, `cryptography`

## Reading notes

- RFC 8731 specifies Curve25519 and Curve448 key-exchange methods for SSH.
- The SSH method name used by the configuration is `curve25519-sha256`.
