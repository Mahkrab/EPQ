# S042 — Tailscale Inc. (2026)

## Full reference:

Tailscale Inc. (2026) ‘MagicDNS’, *Tailscale Docs*.

#### https://tailscale.com/docs/features/magicdns

## Relevance:

Supports the evaluation of a MagicDNS machine name as a more maintainable alternative to placing a literal Tailscale IPv4 address in the SSH client configuration.

## Reliability:

This is official Tailscale product documentation and is authoritative for the current behaviour of MagicDNS. Product behaviour can change between Tailscale versions.

## Tags

`tailscale`, `magicdns`, `dns`, `hostname`

## Reading notes

- MagicDNS automatically registers DNS names for devices in a tailnet.
- A device can be accessed by its machine name instead of its Tailscale IP address when MagicDNS is enabled.
