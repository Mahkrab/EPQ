# S030 — Cloudflare Inc. (n.d.)

## Full reference:

Cloudflare Inc. (n.d.) ‘What is a brute force attack?’, *Cloudflare Learning Center*.

#### https://www.cloudflare.com/learning/bots/brute-force-attack/

## Relevance:

Explains brute-force password attacks and their application to SSH logins, supporting the security rationale for disabling SSH password authentication.

## Reliability:

Cloudflare is an established Internet security provider. The Learning Center gives a suitable secondary explanation of the attack method, although the page also promotes Cloudflare services.

## Tags

`security`, `brute-force`, `password`, `ssh`

## Reading notes

- Brute-force attacks try combinations until a password or key is found.
- SSH logins can be targeted by automated password attempts.
