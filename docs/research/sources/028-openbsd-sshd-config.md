# S028 — OpenBSD Project (n.d.)

## Full reference:

OpenBSD Project (n.d.) ‘sshd_config(5)’, *OpenBSD manual pages*.

#### https://man.openbsd.org/sshd_config

## Relevance:

Provides the configuration reference for OpenSSH server authentication, host keys, key-exchange algorithms and TCP forwarding. It supports the README’s public-key, password-authentication and SSH negotiation terminology.

## Reliability:

This is the official manual maintained by the OpenBSD developers responsible for OpenSSH. It reflects the current development version, so installed-version manuals should be checked before applying exact defaults.

## Tags

`openssh`, `sshd-config`, `authentication`, `key-exchange`

## Reading notes

- OpenSSH supports public-key and password authentication methods.
- `sshd_config` controls authentication methods and permitted key-exchange algorithms.
