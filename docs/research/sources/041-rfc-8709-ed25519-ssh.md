# S041 — Harris and Velvindron (2020)

## Full reference:

Harris, B. and Velvindron, L. (2020) *Ed25519 and Ed448 Public Key Algorithms for the Secure Shell (SSH) Protocol*. RFC 8709. RFC Editor.

#### https://www.rfc-editor.org/info/rfc8709/

## Relevance:

Defines the Ed25519 public-key algorithm used for SSH host and client authentication in the final implementation.

## Reliability:

This is an Internet Standards Track RFC produced through IETF review and published by the RFC Editor. It is authoritative for the `ssh-ed25519` algorithm and its use in SSH.

## Tags

`ssh`, `ed25519`, `public-key`, `authentication`

## Reading notes

- RFC 8709 specifies the `ssh-ed25519` public-key and signature format.
- Ed25519 in SSH is used for signing and authentication rather than encryption.
