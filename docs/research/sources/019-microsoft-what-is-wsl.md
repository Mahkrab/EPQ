# S019 — Microsoft (2025)

## Full reference:

Microsoft (2025) ‘What is the Windows Subsystem for Linux?’, *Microsoft Learn*.

#### https://learn.microsoft.com/en-us/windows/wsl/about

## Relevance:

Explains the WSL 2 layer that hosts the project’s Ubuntu development environment on Windows. Its description of the lightweight virtual machine, Linux kernel, networking and support for Linux services provides background for the remote-access architecture and its behaviour.

## Reliability:

Microsoft develops WSL and publishes Microsoft Learn, making this an authoritative primary source for WSL 2’s architecture and supported capabilities. The page is maintained documentation and states that it was last updated on 21 May 2025.

## Tags

`development-environment`, `wsl2`, `windows`, `virtualisation`

## Reading notes

- WSL runs Linux distributions and command-line tools directly alongside Windows applications.
- WSL 2 uses a real Linux kernel inside a lightweight managed virtual machine.
- Linux services such as `sshd` can run inside a WSL distribution.
