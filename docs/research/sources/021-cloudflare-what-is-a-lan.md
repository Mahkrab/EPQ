# S021 — Cloudflare Inc. (n.d.)

## Full reference:

Cloudflare Inc. (n.d.) ‘What is a LAN (local area network)?’, *Cloudflare Learning Center*.

#### https://www.cloudflare.com/en-gb/learning/network-layer/what-is-a-lan/

## Relevance:

Defines LANs and explains the roles of routers, Wi-Fi and connected devices in a home network. This supports the terminology used to describe the project’s initial local network.

## Reliability:

Cloudflare is an established Internet infrastructure provider. Its Learning Center is a suitable secondary explanation of standard networking concepts, although it also promotes Cloudflare services.

## Tags

`networking`, `lan`, `router`, `wifi`

## Reading notes

- A LAN connects computing devices within a limited geographic area.
- Home LANs commonly connect through one router using Ethernet, Wi-Fi or both.
